===========================================
 PyDedupe: Record linkage tools for Python
===========================================

This software identifies similar records within and between tables, either
database tables or records in a CSV spreadsheet.   We'll introduce the
concepts of record linkage and deduplication, cover the basics of deduplicating
a CSV file, and how to build a record linkage application using the API.
